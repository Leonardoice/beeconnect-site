# Bee Connect Agency — homepage (beeconnectagency.com)

Static site for the Bee Connect Agency homepage. Auto-deploys to Cloudflare Pages on push to `main`.

## Files
- `index.html` — the homepage
- `llms.txt` — AI-crawler readable summary (AIO/GEO)
- `robots.txt` — explicitly allows AI crawlers
- `sitemap.xml`

## Deploy
Connected to Cloudflare Pages (project: beeconnect-site). Push to `main` then auto-build (no build step) then live at beeconnectagency.com.
